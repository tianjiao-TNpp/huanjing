# 环境百词斩

日本大学院 **环境学专攻** 备考词汇 App。基于 [百词斩] 式闪卡，支持手机与电脑，可离线、可安装到桌面/主屏。

纯静态网页（PWA），**无需服务器、无需联网、无依赖**。

## 功能

- **两套词库可切换**：简易（1080 词）/ 进阶（7581 词）。右上角按钮切换。
- **闪卡翻面**：看日语单词 → 点击翻面看读音、释义、相关词、易混词、学习备注；标记「认识 / 不认识」。
- **选择题测验**：四选一，随机正向（词→义）/ 反向（义→词）出题。
- **间隔复习（SRS）**：基于 SM-2 算法记录掌握度，自动安排到期单词复习。
- **按分类 / 研究方向筛选**：12～67 个术语类别，或 78 个研究方向，挑选学习范围。
- **浏览词表 + 搜索**：按单词 / 读音 / 释义搜索，点开看完整词条。
- 学习进度保存在浏览器本地（localStorage），换设备不互通。

电脑端快捷键：闪卡 `空格`翻面、`←/→`(或 `J/K`) 标记；测验 `1~4`选项、`回车`下一题。

## 本地运行

由于浏览器安全限制，**建议用本地服务器打开**（直接双击 `index.html` 在部分浏览器可用，Chrome 可能受限）。

```powershell
# 在本文件夹内执行任一种：
python -m http.server 8000
#   然后浏览器打开  http://127.0.0.1:8000
```

或用 VS Code 的 **Live Server** 插件右键 `index.html → Open with Live Server`。

## 部署到网上（手机随时可用，推荐 GitHub Pages，免费）

1. 把整个文件夹推到一个 GitHub 仓库。
2. 仓库 Settings → Pages → Source 选 `main` 分支根目录，保存。
3. 等 1 分钟，访问 `https://<用户名>.github.io/<仓库名>/`。
4. 手机浏览器打开该网址 → 菜单「添加到主屏幕」即可像 App 一样使用，并可离线。

> 也可用 Netlify / Vercel：直接拖拽本文件夹即可发布。

## 安装为「App」

打开网页后：
- **手机（iOS Safari / Android Chrome）**：分享/菜单 → 「添加到主屏幕」。
- **电脑（Chrome / Edge）**：地址栏右侧的「安装」图标 → 安装。

之后即使断网也能使用（Service Worker 已缓存全部数据）。

## 激活码系统（只有你同意的人能用）

应用启动时会要求输入**激活码**，验证通过才进入；通过后该设备会记住，无需重复输入。

- 配置文件：`access.js`。里面只存激活码的 **SHA-256 哈希**（加了 salt），不存明文，别人看到也读不出码。
- **新增一个人**：
  ```powershell
  python scripts/gen_code.py 张三
  ```
  它会输出①「明文激活码」（私下发给本人）②要粘进 `access.js` 的 `codes` 数组的一行。粘好后重新部署即可。
- **吊销某人**：删掉 `access.js` 里对应那一行（按 label 找），重新部署。该人下次打开就进不去。
- **临时关闭验证**：把 `access.js` 里 `enabled` 改成 `false`。

> ⚠️ 两点务必注意：
> 1. **明文激活码绝对不要写进任何会部署的文件**（包括本 README）。只私下发给本人，自己另存一份。
> 2. 改了 `access.js`（或词库）后，**把 `sw.js` 里 `hj-cache-v2` 的数字 +1** 再部署，否则老用户因离线缓存看不到更新。
>
> 说明：这是纯前端门禁，能挡住普通人随便打开，但拦不住懂技术、直接下载数据文件的人。对备考词库够用；若要严格防破解需配后端。

## 更新词库

词库源文件：`环境学专业入门词库.xlsx`（3 张表：简易单词书 / 进阶单词书 / 研究方向及与单词生成）。
修改后重新生成数据：

```powershell
pip install openpyxl
python scripts/build_data.py
```

会重新生成 `data/*.js` 与 `data/*.json`。

## 目录结构

```
环境百词斩/
├─ index.html           入口
├─ style.css            样式（含深色模式、响应式）
├─ app.js               全部逻辑（无框架）
├─ manifest.json        PWA 清单
├─ sw.js                Service Worker（离线缓存）
├─ icons/               图标（SVG + 192/512 PNG）
├─ data/                由脚本生成的词库数据（.js 供离线加载 / .json 备用）
├─ scripts/build_data.py  xlsx → 数据 的转换脚本
└─ 环境学专业入门词库.xlsx  词库源文件
```
